Projects

3D Game Engine

C++ 3D Game Engine, focused on modern rendering techniques with Vulkan, and modern (C++) engine programming patterns.

Maze Crusade

Assembly 6502 game, made for the NES. Contains a maze generation algorithm and 2 pathfinding algorithms. This project taught me a lot about low level programming & memory management.

Flowfield Pathfinding

A gameplay programming research project, researching how to make pathfinding in games with big crowds more optimal using flowfields. This approach makes adding costs for specific unit types (large simulation games) very easy and can easily be combined with different steering behaviours.

I will continue this project in the near future to further optimise the algorithm, and clean the code up. Additionally I would like to test this in 3D (using my 3D engine project).

Bumpurr Cars

A game project in Unreal Engine. In this project I was responsible for prototyping, coding the score system, hazard system and powerup spawn system and physics. This was a group project so it also refined my soft skills (planning, working in team, testing tasks). Source control was done on Perforce.

Project is currently on-going. - will be updated in the future.
