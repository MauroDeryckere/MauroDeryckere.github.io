Renderer

Rendering pipeline overview

Bindless Rendering

Global index and vertex buffers with draw commands batched via vkCmdDrawIndexedIndirect. Textures stored in a descriptor array.

Instanced Drawing

Tested with 100,000 instances of a 1425-index mesh running smoothly at 60 FPS with room to spare on an RTX 3060.

Deferred Shading

G-buffer based deferred rendering pipeline with a depth prepass to minimize overdraw.

Mesh & Material Loading

Assimp integration supporting all major 3D formats. Meshes split into instanced submeshes with file-based material loading.

Lighting & Tone Mapping

Point and directional lights with configurable exposure. Multiple tone mapping modes.

Debug Rendering

Simple API for drawing lines, spheres, and light visualizations at runtime. Useful for gameplay and physics debugging.

Flight Helmet - tone mapping and material rendering

100,000 instanced meshes

Engine Core

Event System

Delegates with immediate and queued broadcasts, lambda and member function binding, safe delayed unsubscription.

Input System

Action-based input with mapping contexts, keyboard and mouse support, runtime rebinding.

Component System

EnTT-backed ECS with a wrapper providing the engine's interface for entities, components, and queries.

Logging

Multi-level, multi-category logger with console and file output, configurable rotation and color coding.

Profiling

Built-in profiler outputting to Chrome tracing format. Optional Optick integration for detailed analysis.

Timer Manager

One-shot and looping timers, next-tick scheduling, pause/resume/reset with handle-based management.
